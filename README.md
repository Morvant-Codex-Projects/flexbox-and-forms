# Garden Party Invitation

A simple web page that lets users register for a event.

## RSVP Form Planning

An RSVP form that acts as an event invitation that will be sent back to the invitee.

### Inputs

- RSVP
  - Type: Radio
  - Required
- Guest Names
  - Type: Text
  - Required
- Party Size
  - Type: Number
  - Required
- Names of Guests Attending
  - Type: Text
  - Required
- Phone Number
  - Type: tel
  - Call or Text?
